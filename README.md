# Анализ акций IT-гигантов (2025)

Это мой первый проект по финансовой аналитике на Python. В нем я проанализировал доходность акций компаний Apple, Nvidia, Microsoft и Google.

## Что внутри:
* Выгрузка данных через yfinance.
* Аналитическая обработка на Pandas (использование логики SQL: агрегация, оконные функции).
* Визуализация результатов на Matplotlib.
